## Presentation about Rust

Opinions are my own :)

`npm run dev` and `npm run build` compile the Rust wasm demo first, so `wasm-pack` needs to be installed locally.

The generated `cat-demo/pkg` output is committed for this presentation. If you just want to run or bundle the slides without rebuilding the Rust demo, use `npm run dev:prebuilt` or `npm run build:prebuilt`.
